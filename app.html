Enter your Google OAuth Client ID to scan your inbox for subscription receipts automatically. How do I get one?

One-time setup (5 minutes, free): 1. Go to console.cloud.google.com → New Project
2. Search for Gmail API → Enable it
3. Go to APIs & Services → Credentials → Create Credentials → OAuth client ID
4. Application type: Web application
5. Authorized JavaScript origins: add http://localhost:8080
6. Copy the Client ID and paste it below
7. Run the app with: npx serve . -p 8080 then open http://localhost:8080
